# Leap Year Calculator

A simple Java program that prints whether the year input by the user is a leap year or not.

## Features

- Accepts the year input by the user
  
- Calculates whether the year is a leap year or not
  
- Displays the result

##Technologies used

-Java

## Sample Output

2000 is a leap year

-Writing mathematical expressions in Java
